对特征中非int类型的数据进行编码，并划分数据集

In [14]:
LE = LabelEncoder()
x.Platform = LE.fit_transform(df.Platform)
x.Genre = LE.fit_transform(df.Genre)
x.Publisher = LE.fit_transform(df.Publisher)
X_train, X_test, y_train, y_test = train_test_split(x, y, test_size=0.25, random_state=2)

使用交叉验证进行参数调优

In [15]:
#使用交叉验证进行参数调优
Tree = DecisionTreeRegressor(random_state=2)
param_grid = {
     'max_depth': [10, 15],
     #'min_sample_leaf': [30, 50, 100],
     'splitter': ('random', 'best')
}
grid_search = GridSearchCV(Tree, param_grid, scoring='r2', cv=10)
grid_search.fit(X_train, y_train)
print('训练集best_score_: ' + str(grid_search.best_score_))
print('测试集score: ' + str(grid_search.score(X_test, y_test)))  # 分数
print(grid_search.best_params_)  # 最好的超参数
print(grid_search.best_estimator_) #决策树的参数设置

训练集best_score_: 0.9925955193173556
测试集score: 0.879772360102238
{'max_depth': 15, 'splitter': 'random'}

可以发现，决策树最大深度为15、特征划分点选择标准为random时，效果更好，所以使用这一组参数进行训练并预测。

由三个地区的游戏类型占比饼状图可以看出：
(1)在北美地区市场，大家更钟情于动作、运动、设计类型的游戏，而北美又作为国际的主要市场，在推出新游戏时应当着重考虑这三种类型。
(2)在欧洲地区市场，销售量排名较前的游戏类型和北美地区较为相似，也是动作类游戏的受众群体更大。
(3)在日本地区市场，销售的趋势则与北美和欧洲截然不同，角色扮演类游戏在日本占了三分之一的销售比重，可见日本更热爱角色扮演类游戏，所以在推出游戏时应着重考虑这一类型。
总体来说，在三个地区的市场中，动作和运动类型的销量始终比较稳定，稳居前三。
